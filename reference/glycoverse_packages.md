# List all packages in the glycoverse

List all packages in the glycoverse

## Usage

``` r
glycoverse_packages(include_self = TRUE)
```

## Arguments

- include_self:

  Include glycoverse in the list?

## Examples

``` r
glycoverse_packages()
#>  [1] "cli"        "dplyr"      "purrr"      "rlang"      "rstudioapi"
#>  [6] "jsonlite"   "tibble"     "glyclean"   "glydet"     "glyenzy"   
#> [11] "glyexp"     "glymotif"   "glyparse"   "glyread"    "glyrepr"   
#> [16] "glystats"   "glyvis"     "magrittr"   "glycoverse"
```
