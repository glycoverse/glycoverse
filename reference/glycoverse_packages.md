# List all packages in the glycoverse

List all packages in the glycoverse

## Usage

``` r
glycoverse_packages(include_self = TRUE)
```

## Arguments

- include_self:

  Include glycoverse in the list?

## Examples

``` r
glycoverse_packages()
#>  [1] "cli"         "dplyr"       "purrr"       "rlang"       "rstudioapi" 
#>  [6] "jsonlite"    "tibble"      "glyclean"    "glydet"      "glyexp"     
#> [11] "glymotif"    "glyparse"    "glyread"     "glyrepr"     "glystats"   
#> [16] "glyvis"      "glydraw"     "magrittr"    "pak"         "BiocManager"
#> [21] "glycoverse" 
```
