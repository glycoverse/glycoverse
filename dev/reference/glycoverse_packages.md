# List all packages in the glycoverse

List all packages in the glycoverse

## Usage

``` r
glycoverse_packages(include_self = TRUE)
```

## Arguments

- include_self:

  Include glycoverse in the list?

## Examples

``` r
glycoverse_packages()
#>  [1] "cli"         "dplyr"       "purrr"       "rlang"       "rstudioapi" 
#>  [6] "tibble"      "glyclean"    "glydet"      "glyexp"      "glymotif"   
#> [11] "glyparse"    "glyread"     "glyrepr"     "glystats"    "glyvis"     
#> [16] "glydraw"     "pak"         "BiocManager" "checkmate"   "glycoverse" 
```
